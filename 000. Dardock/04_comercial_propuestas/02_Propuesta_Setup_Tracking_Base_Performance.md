# Propuesta — Setup Tracking + Base Performance

## Para quién aplica

Negocios que quieren operar anuncios con lógica de performance, pero hoy no tienen una base técnica ni una estructura mínima confiable para medir y optimizar.

---

## Diagnóstico base

Escalar sin tracking confiable obliga al algoritmo y al equipo a tomar decisiones con señales incompletas. Eso eleva CAC, distorsiona ROAS y ralentiza el aprendizaje.

En este escenario, el problema no es falta de anuncios. El problema es falta de infraestructura para operar performance de forma seria.

---

## Objetivo de esta intervención

Dejar implementada la base técnica y operativa necesaria para que el negocio pueda:

- medir correctamente
- atribuir mejor
- lanzar campañas con criterio
- tomar decisiones sobre datos más confiables

---

## Qué incluye

1. Definición de arquitectura de tracking
2. Configuración o corrección de:
   - Pixel
   - CAPI
   - eventos prioritarios
   - nomenclatura UTM
3. Revisión de match quality y calidad de señal
4. Estructura base de campañas según objetivo
5. Definición de naming conventions
6. Checklist de QA pre-lanzamiento
7. Dashboard base o scorecard de control
8. Documento de implementación y recomendaciones

---

## Entregables

- mapa de tracking
- checklist técnico implementado o validado
- convención de naming
- estructura base de campañas
- scorecard inicial de performance
- backlog de siguientes optimizaciones

---

## Qué no incluye

- gestión mensual continua
- producción masiva de creativos
- CRO profundo
- automatizaciones avanzadas fuera del scope acordado

---

## Dependencias críticas

Esta fase depende de:

- accesos completos
- colaboración del cliente o desarrollador
- plataforma de ecommerce / CRM / marketplace disponible
- confirmación de eventos relevantes de negocio

---

## Duración estimada

- 7 a 15 días hábiles según complejidad del ecosistema

---

## Criterio de éxito

La fase se considera exitosa cuando:

- los eventos clave están definidos y validados
- existe consistencia básica entre fuente, clic y conversión
- Dardock puede lanzar u optimizar campañas sobre una base medible

---
