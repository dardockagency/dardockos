# DARDOCK AUDIT MASTER TEMPLATE

## Objetivo

Este template concentra en un solo sistema maestro los 4 pasos del proceso Dardock:

1. extracción de data
2. análisis
3. publicación para cliente
4. aplicación de cambios

No es solo una auditoría.

Es el puente entre:

- datos
- criterio
- dashboard
- roadmap
- ejecución

---

## Instrucción de uso

Cada cliente debe pasar por este template.

No hace falta llenarlo entero al inicio, pero sí debe respetarse la secuencia.

La lógica es:

- primero capturar
- luego diagnosticar
- luego sintetizar
- luego publicar
- luego ejecutar

---

## Reglas de calidad

1. No hacer data dump.
2. No mezclar síntoma con causa.
3. No recomendar subir presupuesto como primera respuesta.
4. No esconder fallas de tracking o funnel.
5. No cerrar análisis sin siguiente acción.

---

## Salida mínima esperada

Al terminar este template, Dardock debe tener:

1. una auditoría completa
2. una lectura ejecutiva
3. un dashboard cliente
4. un roadmap accionable

---

## Regla final

El valor de Dardock no está en extraer datos.

Está en convertir datos desordenados en:

- claridad
- decisión
- prioridad
- ejecución
